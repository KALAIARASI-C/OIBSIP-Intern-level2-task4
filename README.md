# Login Authentication System

## Objective

The objective of this project is to develop a simple **Login Authentication System** that allows users to register and log in securely. The project demonstrates the concept of user authentication using both **Java (console-based program)** and a **basic web interface using HTML, CSS, and JavaScript**.

## Steps Performed

1. Designed a **console-based authentication system** using Java.
2. Used **HashMap** in Java to store usernames and passwords.
3. Implemented functions for **user registration, login verification, and secured page access**.
4. Created a simple **web interface using HTML and CSS** to simulate a login system.
5. Added input fields for username and password.
6. Implemented **JavaScript functions** for registering users and validating login credentials.
7. Displayed messages such as successful login, registration success, or invalid login.
8. Tested both the **Java console application** and the **web interface**.
9. Uploaded the project files to **GitHub** for version control and project sharing.

## Tools and Technologies Used

* Java (Core Java)
* HashMap (Data Structure)
* HTML5
* CSS3
* JavaScript
* Visual Studio Code / Any Java IDE
* GitHub

## Outcome

The outcome of this project is a **functional login authentication system** that demonstrates the basic concept of user registration and login verification. The Java program shows how authentication works in a console-based environment, while the web interface provides a simple graphical representation of the login system.
